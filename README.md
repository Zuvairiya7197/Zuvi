# Zuvi Studio Portfolio

Premium 3D interactive portfolio website for a freelance graphic designer.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js, React Three Fiber, Drei

## Commands

```bash
npm run dev
npm run typecheck
npm run build
```

## Routes

- `/`
- `/work`
- `/work/[slug]`
- `/services`
- `/about`
- `/blog`
- `/blog/[slug]`
- `/contact`
- `/sitemap.xml`
- `/robots.txt`
